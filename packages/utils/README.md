# `@fixerframework/utils`

Shared foundation utilities for FixerFramework. Zero runtime dependencies.

## Install

```bash
bun add @fixerframework/utils
```

Registry: `https://registry.fixerframework.com` (scope `@fixerframework`).

## Usage

```ts
import { stableStringify, deferred, decodeJwt, serializeCookie } from "@fixerframework/utils";
```
